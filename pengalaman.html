Pengalaman Saya Selama PKL

Proyek A: Pengembangan Aplikasi Web

Saya terlibat dalam pengembangan aplikasi web menggunakan HTML, CSS, dan JavaScript. Dalam proyek ini, saya belajar tentang pengembangan antarmuka pengguna dan pengalaman pengguna (UI/UX). Saya juga menggunakan beberapa framework seperti Bootstrap untuk mempercepat pengembangan tampilan responsif.

  • Membuat desain halaman web yang responsif dengan menggunakan media query CSS.
  • Mengimplementasikan fungsionalitas interaktif dengan JavaScript.
  • Menyiapkan halaman untuk aksesibilitas yang lebih baik bagi pengguna.

Proyek B: Mengoperasikan Bio Finger

Kami belajar cara mengelola atau menghubungkan perangkat fingerprint (Bio Finger) ke komputer untuk pendataan absensi karyawan. Pengalaman ini memberikan pemahaman tentang pengolahan data dalam sistem absensi dan penggunaan perangkat keras dengan software.

  • Melakukan instalasi dan konfigurasi perangkat fingerprint.
  • Mengelola data absensi dan melakukan pemrograman untuk mengintegrasikan perangkat fingerprint dengan sistem database.
  • Memahami proses autentikasi biometrik dan keamanannya.

Proyek C: Pengembangan Aplikasi Desktop

Saya juga terlibat dalam pengembangan aplikasi desktop menggunakan C# dan Visual Studio. Dalam proyek ini, saya belajar bagaimana membuat aplikasi dengan antarmuka grafis (GUI) serta menghubungkannya dengan database menggunakan SQL Server.

  • Membuat tampilan antarmuka pengguna (GUI) dengan Windows Forms.
  • Melakukan koneksi aplikasi dengan database menggunakan ADO.NET.
  • Mempelajari pengolahan data dalam aplikasi desktop, seperti pencarian, penyimpanan, dan pembaruan data.

Proyek D: Pengelolaan Sistem Basis Data

Saya mendapatkan pengalaman dalam mengelola basis data menggunakan MySQL. Dalam proyek ini, saya belajar cara mendesain dan mengelola database untuk aplikasi yang dikembangkan.

  • Mendesain skema database menggunakan diagram entitas-relasi (ERD).
  • Menulis query SQL untuk operasi CRUD (Create, Read, Update, Delete) pada tabel database.
  • Optimisasi query untuk meningkatkan kinerja aplikasi.
