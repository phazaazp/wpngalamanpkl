Halo, Kenali Kami!

Perkenalkan, kami adalah dua siswa yang sedang menjalani Praktik Kerja Lapangan (PKL) di bidang Rekayasa Perangkat Lunak (RPL) di PT. Thasa Indonesia. Kami memiliki passion di dunia teknologi dan bersemangat untuk mengembangkan keterampilan di bidang pengembangan perangkat lunak dan pengelolaan data. Kami berharap pengalaman ini dapat menjadi batu loncatan untuk menjadi profesional yang handal di industri teknologi.

Anggota Tim

  • Phaza Anggie Finola - Siswa Jurusan RPL, fokus dalam pengembangan aplikasi .
  • Meisye Yulia Sukma Raga - Siswa Jurusan RPL, fokus dalam pengembangan aplikasi desktop dan manajemen database.

Tujuan Kami

Selama program PKL ini, kami berharap bisa mempelajari lebih dalam mengenai proses pengembangan perangkat lunak di dunia nyata. Kami juga ingin memperluas keterampilan teknis kami, seperti pengembangan aplikasi web, pengelolaan database, serta bekerja dalam tim pengembangan perangkat lunak. Website ini adalah salah satu hasil dari proyek yang kami kerjakan bersama selama PKL, yang bertujuan untuk berbagi pengalaman dan pengetahuan yang kami peroleh di perusahaan.

Keterampilan yang Kami Kuasai

  • Pengembangan aplikasi web menggunakan HTML, CSS, JavaScript, dan framework terkait.
  • Manajemen dan pembuatan query database menggunakan MySQL dan SQL Server.
  • Pengembangan aplikasi desktop menggunakan C# dan Visual Studio.
  • Kolaborasi tim menggunakan metodologi Agile untuk mengelola proyek perangkat lunak.
  • Pemecahan masalah dan analisis teknis dalam pengembangan perangkat lunak.

Harapan Kami

Kami berharap bisa terus belajar dan berkembang dalam dunia pengembangan perangkat lunak. Melalui PKL ini, kami ingin mendapatkan wawasan yang lebih luas mengenai tantangan dan peluang di industri teknologi informasi. Kami juga berharap website ini bisa menjadi salah satu cara untuk berbagi pengalaman kami dengan orang lain, sekaligus sebagai portofolio untuk perkembangan karier kami ke depan.
